tarkov-stash

This mod for SPT allows you to edit your stash visually. Can't find those USB drives? I got you. You have some item that is not found in raid? Say no more!

Main features

Stash explorer

See all your items in your stash, open any backpack or weapon and see its stats:
This video content might be outdated

Add items

This allows you to add any item of the game, for weapons and armors please use Add preset button.

Add presets

This is the recommended way to add weapons and armors since it includes all the required attachments to work.

Set item to FiR

A simple toggle to mark items FiR.

Restore durability

Kind of useless but if you have few medkits half used you can always restore them to max instead of deleting and adding them again. You can also repair your armors.

Delete items

If you added the wrong item, this might help you :-)

Installation

The mod comes in zip format, just double click it and drag and drop the content of it in your SPT folder. If you had installed a previous version make sure you go to your SPT/user/mods folder and delete the previous one.
